1. Click on an element in the document to see information about that element.
  2. It will show the element, and its classes, like DIV.class1.class2.
  3. It will also show each parent and some children. Click on one of the parents to see that element highlighted in the document.
  4. $('.selector') is the core of all scraping.
  5. If the element you want to get doesn't have a good class or id on it, look for a parent element that does.
  6. Once you find a container, you can use .find(selector) to find children of that element that match a selector.
  7. If you want the text content of an element, us el.strip() which gets the text of the element, but with whitespace stripped and normalized.
  8. .strip() is one of a few extensions that jseeitsaveit.js adds to jquery.
  9. el.link() gets the href or src attribute of an element, resolved.
  10. Another common need is to select a bunch of container elements, and product a list of items from those elements. You can do it like this: 
    var els = $('li.item');
var items = els.collect(function (el) {
  item = {};
  if (! el.find('.required-subitem').length) {
    // Skips this item
    return;
  }
  item.url = el.find('a').link();
  return item;
});
    It's kind of a map pattern. You might want to $.reverse(items) if the order is backwards.
  11. el.clean() returns cleaned-up HTML, removing any inline styles or other suspicious content.
